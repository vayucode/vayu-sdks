# Vayu SDKs

Official client libraries for the [Vayu](https://withvayu.com) API.

| Language | Package | Install | Docs |
|----------|---------|---------|------|
| TypeScript | [![npm](https://img.shields.io/npm/v/vayu-ts?label=vayu-ts)](https://www.npmjs.com/package/vayu-ts) | `npm install vayu-ts` | [typescript/README.md](./typescript/README.md) |
| Python | [![PyPI](https://img.shields.io/pypi/v/vayu-client?label=vayu-client)](https://pypi.org/project/vayu-client/) | `pip install vayu-client` | [python/README.md](./python/README.md) |
| Go | [vayu-sdks/go](https://github.com/vayucode/vayu-sdks/tree/main/go) | `go get github.com/vayucode/vayu-sdks/go` | [go/README.md](./go/README.md) |

Each SDK has its own documentation with setup instructions, usage examples, and API reference. See the links above.
