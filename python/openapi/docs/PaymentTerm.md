# PaymentTerm

The payment term of the product. Can be prepayment or postpayment. If not provided, the payment term will be determined by the pricing model.

## Enum

* `PREPAYMENT` (value: `'Prepayment'`)

* `POSTPAYMENT` (value: `'Postpayment'`)
